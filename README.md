# MISS College Website Redesign

A modern, premium redesign of the MISS College (Madurai Institute of Social Sciences) website with world-class UI/UX, responsive design, and comprehensive content coverage.

## 🎨 Design Features

- **Premium Modern Design**: Inspired by Stanford, Harvard, MIT, and Oxford
- **Glassmorphism Effects**: Modern UI with glass-like transparency
- **Responsive Layout**: Fully mobile-responsive across all devices
- **Smooth Animations**: Framer Motion animations for engaging interactions
- **Accessibility**: WCAG-compliant design for all users
- **Professional Color Palette**: Navy, Blue, Gold, and Emerald theme

## 🚀 Tech Stack

- **Framework**: Next.js 14 (React)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Components**: Custom reusable component system
- **Design**: Mobile-first responsive design

## 📦 Installation

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   npm start
   ```

4. **Access the Site**
   - Open [http://localhost:3000](http://localhost:3000) in your browser
