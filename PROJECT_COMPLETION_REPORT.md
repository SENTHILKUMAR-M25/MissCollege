## рџЋ“ CUSTOMIZATION GUIDE

### To Update Content
1. Edit individual page files in `app/` directory
2. Update component props in each page
3. Modify text, dates, and information

### To Change Colors
1. Edit `tailwind.config.js`
2. Update color values in CSS custom properties
3. Colors referenced in entire site will update automatically
